使用Git submodule备份博客提交记录

发表于 2021-05-19 分类于 Hexo ， Next 阅读次数： Waline：
本文字数： 129 阅读时长 ≈ 1 分钟

当博客和博客发布都在git上时使用Git submodule备份博客提交记录

适用场景

博客的备份和发布都使用git管理

方法

  1. 移除目录中的 .deploy_git
  2. 运行 git submodule add <博客发布git仓库> .deploy_git
  3. 提交 commit
